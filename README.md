# AI Resume Analyzer

This project analyzes uploaded PDF resumes and extracts skills, gives a score, and suggests job roles.

## Technologies
- Python
- Flask
- SQLite (optional)
- PyPDF2
- HTML/CSS
- Vercel

## Run Locally

Install dependencies:

pip install -r requirements.txt

Run:

python api/index.py

Open:
http://127.0.0.1:5000

## Deploy
Upload to GitHub and import into Vercel.
